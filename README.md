# GRX STRING LIGHTS

3D printing project for decorative string light covers with Voronoi pattern, Granada-inspired design.

## Current Files

| File | Description |
|------|-------------|
| `Voronoi-Granada-Light.3mf` | Optimized lightweight version for premium finish |

## Print Settings (Optimized)

| Parameter | Value |
|-----------|-------|
| Layer height | 0.20mm |
| Line width | 0.45mm |
| Wall loops | 2 |
| Top/Bottom layers | 3 |
| Infill density | 10% |
| Infill pattern | Gyroid |
| Outer wall speed | 150mm/s |

**Target weight:** <12g per unit

## Changelog

### v2 - Lightweight Optimized (2024-12-09)
- Reduced layer height from 0.28mm to 0.20mm for smoother finish
- Changed infill from 15% grid to 10% gyroid for weight reduction and strength
- Increased top/bottom layers from 2 to 3 for better surface quality
- Reduced outer wall speed from 200 to 150mm/s for premium finish
- Wider line width (0.45mm) for stronger walls with less material
- **Target:** ~40% weight reduction (from ~19g to <12g)

### v1 - Initial Release
- Added `Voronoi-Granada.3mf` - Initial Voronoi pattern design (archived in git history)
